### Instalação

```bash
# Faça o clone do repositório

# Modifique as variáveis de ambiente em um arquivo .env
  AUTH_SECRET=
  PORT=

# Faça a instalação das depêndencias
  npm i

# Rode as migration
  npm run migrate

# Executando o projeto no ambiente de desenvolvimento
  npm run dev
```
